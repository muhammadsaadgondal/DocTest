# Getting Started

Welcome to DocTest! This guide will help you get up and running quickly.

## Prerequisites

Before you begin, ensure you have the following installed:

- Git (version 2.0 or higher)
- Node.js (version 14.0 or higher)
- npm or yarn package manager

## Installation

### Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/muhammadsaadgondal/DocTest.git
   cd DocTest
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the project:
   ```bash
   npm start
   ```

## Verify Installation

To verify that everything is installed correctly, run:

```bash
npm test
```

If all tests pass, you're ready to go!

## Troubleshooting

### Common Issues

**Issue: Installation fails**
- Solution: Make sure you have the correct Node.js version installed

**Issue: Tests fail**
- Solution: Clear your node_modules and reinstall dependencies
